# 🌍 Travel Diaries

Travel Diaries is a web application where users can share travel experiences by creating posts with descriptions and images. Other users can explore posts, leave comments, and upvote their favorite travel stories.

The goal of this project is to create a simple community platform where people can document and discover travel experiences.

---

## ✨ Features

- Create travel posts with **title, description, and image**
- View posts in a feed
- Upvote posts
- Comment on posts
- Edit or delete your own posts
- Search and sort posts
- Responsive user interface
